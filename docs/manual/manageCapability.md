# Manage Capability

A capability is the credentials that is required when you need to log in to access website data.

### How to add a capablity
* Click the "Capability" menu on the left
* Clcik the "Add" button
* Input the keyword of capability and Enter
* Click the name of capability that you want to add
* Click the "Use selected account category" button
* Select the "Login Method" if multiple login methods are supported
* Input the content(such as username/password) if Login Method is "auto"
* Edit other attributes of capabiltiy if needed
* Click the "Add" button

### How to manage capabilities
* Select the capabilities you want to manage.
* Click "More" and "Delete" / "Lock" / "Unlock" / "Disable" / "Enable"

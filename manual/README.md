# Quick Start - User Guide

Welcome to register and use LetsScrapeData web scraping tool.

### Sign up
* If you haven't registered yet, please [register for a LetsScrapeData account](https://web.LetsScrapeData.com/register) first (need an email address)

### Installation on Windows
* Download the [LetsScrapeData installer](https://www.LetsScrapeData.com/soft/LetsScrapeData_setup.exe) for Windows.
* Once it is downloaded, run the installer.

### Log in
* By default, the app starts automatically. Open "Start LetsScrapeData" in the Windows Start menu if the LetsScrapeData app hasn't started yet
* Open "[Open LetsScrapeData Page](http://127.0.0.1:8090/)" in the Windows Start menu
* Log in with your username/email address and password

### Add new schedule
* Click the Schedule menu
* Click the Add button
* Search the Template Id (tid)
* Select the desired template
* Click the "Use selected template" button
* Edit the Name / Timing Configuration if needed
* Input parameters if needed
* Click the "Save & Start Schedule" button
* Refer to the video on [how to scrape data using LetsScrapeData](https://youtu.be/SO5aSVmNhlQ)

### View the scraped data
* Click the Schedule menu
* Select a template in the list of templates
* Select the data table in the upper-right tree
* The scraped sample data will be displayed in a table in the lower right corner
* The [scraped data can be exported](/manual/exportData) if needed
* The [scraped data can be synchronized](/manual/syncData) to a relational database if needed
* Refer to the video on [how to scrape data using LetsScrapeData](https://youtu.be/SO5aSVmNhlQ)

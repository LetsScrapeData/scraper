# Extension Commands

All commands in LetsScrapeData extension are listed as below:

ID | Title | Keyboard Shortcuts
---------|----------|---------
lsd.clearHighlightedElements | Clear Highlighted Elements	| 
lsd.commitTemplate | Commit Template Config & Lang	| Ctrl + Alt + C
lsd.executeAllActions | Execute All Actions	| Ctrl + Alt + A
lsd.executePartActions | Execute Part Actions | Ctrl + Alt + P
lsd.extractSomeColumns | Extract Some Columns | Ctrl + Alt + F / Shift + F (Vim)
lsd.getNewTabname	| Get New Tabname | 
lsd.getTemplCfg | Get Template Config |
lsd.getTemplLang | Get Template Lang |
lsd.saveTemplCfg | Save/Add Template Config | Ctrl + Alt + S
lsd.saveTemplLang | Save Template Lang | Ctrl + Alt + L
lsd.searchDomains | Search Domains | 
lsd.searchTabnames | Search Tabnames | 
lsd.searchTemplCfgs | Search Template Configs | 
lsd.showParentXml | Show Parent Xml | Ctrl + Alt + T / Shift + T (Vim)

* Vim: valid when Vim extension is installed and in normal mode.
